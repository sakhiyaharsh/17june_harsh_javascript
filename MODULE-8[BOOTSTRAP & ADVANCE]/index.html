QUE:1What are the advantages of Bootstrap ?

2. Grid System: Bootstrap's grid system is highly flexible and responsive, allowing you to create complex layouts that adapt to different screen sizes and orientations.

3.Responsive Design: Bootstrap is built with responsive web design in mind. It uses a responsive grid system that automatically adjusts the layout and content of a website to fit different screen sizes, making your site mobile-friendly without much effort.

QUE:3What are the default Bootstrap text settings ?

-> Font Family

-> Font Weight: By default, Bootstrap uses a font-weight of 400 for normal text and 700 for bold text.

Line Height: The default line height is 1.5, which helps with readability and spacing.

Text Color: Bootstrap sets the default text color to the browser's default, typically black.

Text Alignment: Text is aligned to the left by default.

Text Decoration: Links are set to have no underlines by default and change color on hover.

Text Transform: Bootstrap sets text-transform: none; by default, meaning text is displayed as entered.

Heading Styles: Bootstrap provides styles for heading elements (h1 to h6) that include font size, line height, and margin-bottom adjustments.

QUE:4What do you know about the Bootstrap Grid System ?

1-Column Grid: Bootstrap's grid system is based on a 12-column layout. You can create layouts by dividing the available space into 12 equal-width columns. These columns can be combined in various ways to create the desired structure for your web page.

2-Responsive Design: Bootstrap's grid system is inherently responsive. It uses CSS media queries to automatically adjust the layout and column widths based on the screen size.

QUE:5What is the difference between Bootstrap 4 and Bootstrap 5 ?

jQuery Dependency:
Bootstrap 4: Bootstrap 4 had a dependency on jQuery for some of its features and components, such as dropdowns and modals.

Bootstrap 5: Bootstrap 5 removed the jQuery dependency, making it lighter and more modern. JavaScript functionality in Bootstrap 5 is now implemented using native JavaScript.

New Features and Components:
Bootstrap 5 introduced new features and components, such as the offcanvas component for creating off-canvas navigation, and the toast component for displaying non-blocking notifications. Some components from Bootstrap 4 were also refined and improved in Bootstrap 5.

Smaller File Size:
Due to the removal of jQuery and other optimizations, Bootstrap 5 generally has a smaller file size compared to Bootstrap 4.

QUE:8/b> In Bootstrap 4, what is flexbox ?

-> In Bootstrap 4, flexbox, short for "flexible box," refers to the use of the CSS Flexbox (Flexible Box Layout) model to create responsive and flexible layouts for web components. Flexbox is a layout system that allows you to distribute space and align content inside a container, even when their sizes are unknown or dynamic.

-> Bootstrap's integration of flexbox makes it easier to create responsive and flexible layouts without the need for extensive custom CSS.
