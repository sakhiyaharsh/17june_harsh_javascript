QUE:14What are all the types of Pop up boxes available in JavaScript ?

-> In JavaScript, you can create various types of popup boxes or dialog boxes to interact with users.

1. Alert Box:

-> The alert() method displays a simple dialog box with a message and an "OK" button.

-> It is primarily used to provide information or warnings to the user.


2. Prompt Box:


The prompt() method displays a dialog box that prompts the user to enter some text.

3. Confirm Box :


The confirm() method displays a dialog box with a message and "OK" and "Cancel" buttons.
