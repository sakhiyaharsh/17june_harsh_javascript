QUE:5 What are undeclared and undefined variables ?

UNDECLARED VARIABLE:

-> Undeclared variables are those that have not been declared or defined in the current scope..

-> undefined variables are those that have been declared but not given a value.


EXAMPLE

console.log(x); // Error: x is not defined

-> in this example x is not defined in the current scope....


UNDEFINED VARIABLE:

-> When a variable is declared without assigning an initial value, it is assigned the default value of undefined..

-> Undefined variables in JavaScript are variables that have been declared.

EXAMPLE

let y;
console.log(y); // undefined
