QUE:8 What is === operator ?

-> In JavaScript, the === operator is the strict equality operator. It is used to compare two values for equality, taking into account both the value and the data type..

-> When you use the === operator to compare two values, JavaScript checks if the values are equal and have the same data type. If both the value and the data type match, the result is true. If either the value or the data type differs, the result is false.
